Bash shell commands sheet

#SCP Command#

scp local_file_dir usename@ip_address:remote_dir

第一个参数为 ./本地的文件名，第二个参数为 用户名@输入IP地址或者url:/远端的文件名绝对地址
*注：传输文件夹，scp加 -r
远端传至本地同理
#Basic terminal shortcuts

CTRL A = To start of line
CTRL E = To the end of the line
CTRL U = Delete left of cursor 
CTRL K = Delete right of cursor
CTRL W = Delete word on the left 
CTRL Y = Paste (after CTRL U,K or W)
CTRL L = Clear the terminal
CTRL Z = Stop the current command

#Basic Terminal Navigation

ls = list all files
cd folder name  = change directory
cd / = go to root
cd .. = go to one up folder
pwd  = print working directory

#Basic File Manipulation

mkdir = create a new folder
mkdir myfolder
mkdir mufolder/yj/pics/..

mv file.txt Documents/ = move to a folder
mv <foldername1> <foldername2> = move folder in folder
mv filename.txt filename2.txt = rename file
 
rm <filename> .. = delete file (s)
rm -i <filename> .. = ask for confirmation each file

touch <filename> = create or update a file
